# Getting Started

Welcome! Create pages under `content/` using MDX. Examples:

- `content/index.mdx` → homepage at `/index.html`.
- `content/sitemap.mdx` → sitemap at `/sitemap.html`.
- `content/docs/getting-started.mdx` → `/docs/getting-started.html`.

## Next Steps

- Edit `content/_layout.mdx` to customize header/footer.
- Add more docs in `content/docs/`.
